# Samsung Store Clone 📱

A modern, fully responsive e-commerce web application inspired by the official Samsung Store. Built with React and Vite, this project demonstrates advanced UI/UX principles, dynamic state management, and seamless interactive components to deliver a premium shopping experience.

## 🚀 Features

* **Premium UI/UX:** High-quality, responsive design mirroring the premium feel of Samsung's official website using Tailwind CSS.
* **Auto-Sliding Product Carousel:** Interactive product showcase with auto-play functionality, navigation arrows, and pagination dots.
* **Dynamic Product Pages:** Detailed individual product pages featuring dynamic pricing and color selection.
* **Shopping Cart System:** Fully functional cart architecture (Add/Remove items, subtotal calculation, and order summary) powered by the React Context API.
* **Authentication UI:** Simulated Sign-In and Sign-Up user flows with seamless form toggling.
* **Immersive Hero Section:** High-performance HTML5 background video integration for flagship product highlights.

## 🛠️ Tech Stack

* **Frontend Framework:** React.js powered by Vite
* **Styling:** Tailwind CSS
* **Routing:** React Router DOM
* **Icons:** Lucide React
* **State Management:** React Context API
* **Deployment:** Vercel
